☕ Sip Happens Café - Feedback App

This is a simple feedback collection application developed with React and Vite. It allows users to track feedback (Good, Neutral, Bad), calculates statistics, and persists data using local storage.

🚀 Features
Real-time Statistics: Calculates total feedback and positive feedback percentage.

Persistence: Uses localStorage to save data across page reloads.

Conditional Rendering: Statistics and the Reset button only appear when there is at least one feedback.

Component-Based Architecture: Built with modular components and CSS Modules.

🛠 Tech Stack
React (Functional Components & Hooks)

Vite (Build Tool)

CSS Modules (Scoped Styling)

Prettier (Code Formatting)

📂 Project Structure
Each component is located in its own folder within src/components/, containing a .jsx file and a .module.css file.
